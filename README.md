# Titanic Data Cleaning using Pandas

## Project Overview
This project demonstrates data cleaning and preprocessing using the Titanic dataset with Pandas.

## Features
- Load dataset
- Inspect data
- Handle missing values
- Remove duplicates
- Drop columns with excessive missing values
- Save cleaned dataset

## Technologies Used
- Python
- Pandas
- Google Colab

## Files
- Titanic_Data_Cleaning.ipynb
- titanic_raw_dataset.csv
